# FO’X — What to Upload to ChatGPT Project

## В Project Instructions вставить

Скопировать весь текст из файла:

```text
00_PROJECT_INSTRUCTIONS_PASTE_TO_CHATGPT_PROJECT.md
```

и вставить в поле инструкций проекта.

## В файлы проекта загрузить

Обязательно:
- этот архив / эти `.md` файлы;
- текущий рабочий `index.html`;
- текущий Apps Script backend банкетов;
- текущий XLSX / Google Sheets export стока;
- Apps Script код стока;
- скрин или описание структуры Google Sheets банкетов;
- скрин или описание Cloudinary preset.

Желательно:
- старые версии `index.html`, если нужно сохранять историю;
- changelog версий;
- инструкции деплоя;
- список будущих задач.

## Как называть чаты внутри проекта

Примеры:
- `Банкеты — v13 редактирование`
- `Банкеты — баг с фото`
- `Mini App — дизайн`
- `Сток — формулы`
- `Сток — Apps Script`
- `Deploy — GitHub Pages`
- `Deploy — Apps Script`

## Как начинать новый чат

Скопировать:

```text
Работаем внутри проекта FO’X. Используй проектные инструкции и файлы проекта. Текущая рабочая версия банкетов: v12 Cloudinary READY. Не ломай рабочую систему, сначала предложи минимальный план.
```
